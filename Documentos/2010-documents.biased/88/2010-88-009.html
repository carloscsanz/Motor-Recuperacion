< body >

<body> - defines the body of the document

Syntax

<body> ... </body>

Attributes

Body Attributes

These attributes are specific to the <body> tag.

  • alink - default active link color
  • background - background image
  • link - default link color
  • text - default text color
  • style - inline css rule
  • vlink - default visited link color

CSS Body Attributes

These body attributes are used in combination with CSS.

  • class - css class selector
  • id - css id elector
  • style - inline css style

Other Attributes

  • bgcolor - background color

Description

Examples

Body Background Color

background-color (CSS)

<html>
    <head>
        <title>body  background-color example </title>
    </head>
    <body style="background-color:black;">
    </body>
</html>

bgcolor (HTML)

bgcolor is the old way of declaring a background color with the <body> tag.

<html>
    <head>
        <title> HTML Body Background Color Example </title>
    </head>
    <body bgcolor="black">
    </body>
</html>
