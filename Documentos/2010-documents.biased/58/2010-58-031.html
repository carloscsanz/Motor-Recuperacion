From Microsoft:

QUOTE
Word 97 for Windows/Word 98 Macintosh Mswrd832.cnv 278,528 Used by other Microsoft applications to open Word 97-2000 files.


The file Mswrd832.cnv is used by other MS applications to open Word files.
When you use a program such as WordPad to open a Word document, the convertor tries to translate the document into something you can read. In your case, the document couldn't be translated. That's the reason you received the error message:
QUOTE
wordpad MSWRD832.CNV has performed an illegal operation and will be shut down.


The cause(from MS):

QUOTE
CAUSE
WordPad is a simple Word processor that is only compatible with the following formats:
� Rich Text Format (*.rtf)
� Word Document (*.doc) - Word 6.0 format only
� Text Document (*.txt)
� Windows Write (*.wri)


To be able to read the file, get your friend to send the document back to you in Rich Text Format -rtf. There were some other suggestions in the link I provided in my last post.

Also, there is a Word Viewer that you can download:

Word Viewer

It is supported on XP and 2000 operating systems.
