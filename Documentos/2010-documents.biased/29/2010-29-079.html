Open .ODF file in Microsoft Office – Sun ODF Plugin for Microsoft Office

1 May, 2009, 2:27 am


The ODF (Open Document format) is a file format used by OpenOffice and StarOffice for electronic office documents such as spreadsheets, charts, presentations and word processing documents.

If you want to open, read, edit and save ODF files in Microsoft Office, you should download and install Sun ODF Plugin for Microsoft Office.

The plugin works with Microsoft Office 2007 (Service Pack 1 or higher), Microsoft Office 2003, XP and Microsoft Office 2000….

Sun ODF Plugin gives users of Microsoft Office Word, Excel and PowerPoint the ability to read, edit and save to the ISO-standard Open Document Format (ODF).

Don’t worry, this plugin is easy on setup and use, just download and install it. To make sure that this plugin is installed correctly open Microsoft Office (e.g Microsoft Word) Tools > Templates and Add-Ins if odfaddin.dot exist on “Global templates and Add-Ins” this plugin successfully installed.

Download: http://www.sun.com/software/star/odf_plugin/get.jsp

System Requirements

Computer and Processor: Personal computer with a Pentium compatible Processor
Memory: 256 MB of RAM (512 MB recommended)
Hard Disk: 100 MB available hard disk space
Operating System: Microsoft Windows 2000 (Service Pack 2 or higher), XP or Vista
Microsoft Office Software: Microsoft Office 2000, Office XP, Office 2003, Office 2007 (Service Pack 1 or higher) or the equivalent stand-alone version of Microsoft Office Word, Excel or PowerPoint
