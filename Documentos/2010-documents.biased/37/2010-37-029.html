- As u can see the main difference between the cards 7900gt / gtx is the working clock frequency and for some card the additional  256 mb memory.lets remind

1) that the 7900gtx has 1.2 nano second memory and the 7900gt has 1.4 only witch correspond to 1400 mhz.

2) the 7900gtx has 512 mb memory and the 7900gt has usualy 256mb.

3) the cooling core / memory cooling used in the 7900gtx is a lot better than the one in 7900 gt

4) and last the most importatnt to us is the core working vcore .the 7900 gtx work with 1.4 v core and the 7900 gt has 1.2 only.

so basiquely if we crank the 7900gt vcore to 1.4 we should get the 650 MHz in the core :-) .

the v core mod Can be done very easily with a conductive pen, connecting those 2 points by the yellow line. It will boost the voltage to 1.4v from the original 1.2v, giving it the same potiential as a GTX. Many have reached GTX speeds and beyond by doing this mod. we sould return to the overclocking later on this review.
