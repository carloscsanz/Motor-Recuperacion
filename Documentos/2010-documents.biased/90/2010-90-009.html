Performance - Application Benchmarks

ScienceMark 2.0

ScienceMark 2.0 is a suite of 6 benchmarks that each use scientific algorithms to model a different aspect of real world performance demands. Descriptions of each benchmark, along with the average score of 3 trials follows.

Mol Dyn - 785
Molecular Dynamics is a simulation method that takes the classical potentials plus the electronic Schrï¿½dinger equation of atomic particles and iteratively calculates forces, velocities, and positions through time. The time averaged result yields physical properties such as energy, potential energy, and kinetic energy.
Primordia - 692
This code calculates the quantum mechanical Hartree-Fock orbitals for each electron in any element of the periodic table.
Cryptography - 918
Uses AES cipher to test encryption.
STREAM - 559
Sustainable memory bandwidth.
Mem Bench - 884
Memory Benchmark is a synthetic benchmark that uses many different algorithms to copy integer data from one vector to another vector, to determine memory latency and bandwidth.
BLAS/FLOPS - 951
The Basic Linear Algebra Subroutine is a test of large matrix multiplication speed. It calculates vector dot product, scalar times vector X plus vector Y, matrix-matrix multiplication, and matrix-vector multiplication. It reports the peak floating point operations per second.

The overall score is a weighted average of the 6 benchmarks, which is normalized to 1000 for a baseline system with dual Opteron 246 (2.0 GHz, 1024 MB L2 cache) with 4 GB registered PC2700 DDR RAM, on a non-NUMA motherboard. PC Magazine Winstone 2004

PC Magazine Business Winstone 2004 version 1.0.1 is a set of scripted tests of various business applications, including Microsoft Office XP, WinZip 8.1, and Norton AntiVirus Professional Edition 2003. The Multitasking test uses the same applications as the Business Winstone test, but runs some of them in the background. Multimedia Content Creation Winstone 2004 tests with Adobe Photoshop 7.0.1, Adobe Premiere 6.50, Macromedia Director MX 9.0, Macromedia Dreamweaver MX 6.1, Microsoft Windows Media Encoder 9 Version 9.00.00.2980, NewTek LightWave 3D 7.5b, and Steinberg WaveLab 4.0f. Test scores are reported as an average of 4 trials, with a disk defragment and reboot between each trial. Tests were run with background applications closed/suspended.

  • Business Winstone 2004: 19.1 Winstone units
  • Business Winstone 2004 Multitasking test: 1.92 Winstone units
  • Multimedia Content Creation Winstone 2004: 26.7 Winstone units

These scores are relative to a base computer system, which is a Dell Dimension XPS B1000r with Pentium !!! 1GHz CPU, 256 MB RAM, nVidia GeForce DDR, 30GB Seagate Barracuda Ultra ATA/66 7200 rpm, and MS Windows XP SP1. The base system is normalized at 10.0 on the Business Winstone and Multimedia Content Creation Winstone tests, and at 1.00 on the Multitasking test. For example, a system with a score of 20 Winstone units completes the benchmark test in one half the time of the base system.

PCMark 05

Futuremark PCMark05 Build 1.1.0 is another application-based benchmark to test overall PC performance. It runs portions of real applications rather than installing whole applications or using specifically created code. The 11 tests in the system test suite are each designed to represent a certain type of PC usage: Audio compression, Video encoding, Text edit, Image decompression, File compression, Memory latency, Hard drive - Virus scan, File encryption and decryption, Hard drive usage, Hard drive - Windows Startup, 2D - Transparent Windows, 2D - Graphics memory, 3D - Pixel shader, Web page rendering, Multithreading, and Physics calculation and 3D. The test score is an average of 3 trials using the benchmark default settings, with a hard drive defragmentation and a reboot between each trial. Tests were run with background applications closed/suspended.

  • PCMark05: 2003

Cinebench 9.5

Cinebench 9.5 tests raw processing speed and graphics, with particular attention to rendering, shading, and lighting performance. Cinebench uses Cinema 4D's software shading algorithm, and also graphics hardware functions such as OpenGL acceleration. It is multithreading and multiprocessor aware, although that will not be important for this particular system. The benchmark generates 4 test scores. The test scores below are averages of 3 trials, with a reboot between each trial. The included ATI 2D driver version 6.14.10.6543 was used.

Benchmark Test Benchmark Description Score
CPU Rendering Test of processing speed. The time to render a scene is measured and converted to a normalized score where 100 equals the performance of a Pentium 4 1.0 GHz processor. 294
Cinema 4D Shading Graphics shading. This test uses Cinema 4D's own software shading algorithm. The result is converted to a normalized score where 100 equals the performance of a Pentium 4 1.0 GHz processor. 301
OpenGL software Graphics with OpenGL assist. The lighting is handled by Cinema 4D's software algorithms. Shading and transformation is handled by the graphics hardware. 752
OpenGL hardware Graphics chip OpenGL acceleration. Cinema 4D merely transfers the position of light sources and geometry. A modern GPU will run this test much faster than the software-based algorithms. 419

Note that the best graphics performance was obtained with the OpenGL software bench, which indicates that the OpenGL hardware acceleration from the ATI integrated graphics GPU is not particularly good.
