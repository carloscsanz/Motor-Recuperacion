LameXP is an Encoder Front-End for the popular LAME MP3, Ogg Vorbis and Nero AAC codecs and allows you to convert your audio files to MP3, OGG, Wave, AAC and FLAC formats. It provides an easy to use interface that allows you to drag and drop files, select encoding options, set metadata and quality settings. The program can also create a playlist (.m3u) and save it along with the converted files. Supported input types are Wave, MP3, MP2, OGG/Vorbis, FLAC, MP4/AAC, AC3, DTS, WavPack, Musepack, TTA, Speex, Monkey`s Audio, Shorten, ALAC and WMA. LameXP also reads M3U, PLS and ASX playlists as well as Cuesheets.

Ogg Vorbis is a completely open, patent-free, professional audio encoding and streaming technology that sounds better than MP3 and is gaining in popularity. WinVorbis brings the full OGG Vorbis toolkit in one integrated Windows interface and acts as a graphical front-end for VorbisComment, OggEnc, OggDec and VorbisGain.
