How Can I Force Word Pad to Open an .RTF Document?

WordPad is a basic streamlined word-processing program for Microsoft Windows computers. WordPad is designed to work with TXT (text), RTF (Rich Text Format), and other basic word processing formats. Sometimes you may want to open an RTF file in WordPad but find it insists on opening in Microsoft Word or another more advanced program. You can force the document to open for editing in WordPad with little stress.

    Using WordPad's File Menu

  1. The easiest way to open RTF files within WordPad is to do so through WordPad's File menu. Open WordPad and go to the File menu in the top left-hand corner of the application. Select "Open" to bring up a dialogue box where you can navigate to the file to be opened within WordPad. Simply double-click the file or click "Open" to open the RTF for reading or editing in.

    2. Opening From Windows Explorer

  2. To simultaneously start WordPad running and open the file to be edited in WordPad, right-click on the file within Windows Explorer. A pull-down menu will offer a variety of actions to be performed on the file. Move the mouse pointer to the "Open With..." option. A new dropdown menu will open offering a selection of programs to open the file in. If WordPad is among these programs, select this option. If Windows does not give the option to open in WordPad, select the "Choose Program..." option. Navigate to WordPad's location on your hard drive (WordPad is usually located in the "Program Files\Accessories" directory) and select the program. The file will now open in WordPad.

    3. "Open in WordPad" Registry Hack

  3. This is an advanced technique that should only be performed by users who know their way around the computer registry. Editing the registry can permanently damage your software. To add the option to open RTF files with WordPad by default, go to "Run" in the Start menu and type "regedit." In the Registry Edit application that opens, navigate to the "HKEY_CLASSES_ROOT\" tree. Navigate to the "RTF" extension to open this tree. Right-click on the "Shell" option and select "New" and then "Key." Change the name of the key to "WordPad." Right-click on this key to create another new key and name it "Command." Double-click on the "Default" option to edit the key's function. Set the key's value to the location where WordPad is located, adding "%1" at the end. Typically this will mean entering the following text:

    C:\Program Files\Accessories\WORDPAD.EXE. "%1"

    The "%1" instructs Windows to open the currently selected file in WordPad. Click "OK" to set this value as an option in Windows.
