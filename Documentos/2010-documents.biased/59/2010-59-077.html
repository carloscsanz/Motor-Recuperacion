How to achieve POP3 e-mail by c# program
You need to the client and server connectivity. However, the client to connect to the server, pay attention to the port set to the default POP3 protocol 110. After the success of the client connect to the server, the server will return the following information: + OK Character + OK is the POP3 protocol to return information. It is not SMTP protocol response message as a rich and varied with the number that only two: + OK or-ERR. One, + OK, said a successful connection, while the-ERR said connection failed. Next, the client enter the USER <user name " This command tells the server to your user name. Note that some servers are case-sensitive letters. The server returns + OK, the client-side input PASS <password " After the server returns a + OK. Check that and reply.
