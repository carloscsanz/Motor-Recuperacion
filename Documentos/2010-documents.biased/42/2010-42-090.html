Windows XP Service Pack 3 Overview

Windows Vista got its first service pack last month (March 2008) and this means XP SP3 is next, after almost four years of waiting. Windows XP Service Pack 2 was released in August 2004. Microsoft says that XP SP3 is a "..a rollup that includes all previously released updates for Windows XP, including security updates, out-of-band releases, and hot fixes. It contains a small number of new updates, but should not significantly change the Windows XP experience."

In other words, Microsoft is saying we have taken xp sp2 added all the patches and updates we have released over the last 3 + years, included some new ones and we are calling this service pack 3. Humph, four years for that, not much I would say. We could speculate it is because of the poor sales performance of Vista. Since Microsoft has not released any sales figures for Vista, I cannot really comment on that. I know the following :-

  • Microsoft has extended the date you can get XP retail or on a new PC from January 2008 to June 2008.
  • System builder partners can offer XP until January 30, 2009
  • Ultra Low Cost Pcs can be purchased with XP Starter Edition only until the later of June 30, 2010, or one year after general availability of the next version of Windows
  • Windows 7, the version to replace Vista is slated for release in 2010 or later.
  • Microsoft provides support (ie updates) for a service pack at least 12 months after its release.
  • XP Service Pack 3 is the last service pack for Windows XP.

Of course, any Monday morning, Microsoft can make a new announcement changing all that. This is why I will be updating this page frequently with any new information I get my hands on, especially a release date which is anytime between now and June 30, 2008. Windows XP SP3 news

As at April 4, 2008

  • Windows XP SP3 Release Candidate 2 has been sent to beta testers. It includes support for HD Audio (helpful for gamers and DVD movie viewing) and some minor fixes.
  • Service Pack 3's size is expected to be from 70Mb to 580mb. It varies depending on how much updates you have been downloading and installing over the years. Now would be a good time to catch up on that.
  • You will be able to install service pack 3 on top of service pack 1 or 2.

As at April 12, 2008

  • Some sites have performed tests between Windows XP SP3 and Vista SP1 and have proclaimed XP SP3 as faster. These test were performed on a pre-release version (XP SP3 RC2) so this is not official yet.

As at April 16, 2008

  • There are sites announcing the release date of XP SP3. The most specific is a OEM (Dell,etc) release on April 21, 2008. A release to the general public on April 29, 2008 and a Windows Update release (you can't fight it anymore) by June 30, 2008. Remember this is not an official Microsoft release date. It is probably a leak though it may be accurate. Check back by the end of the week for an update.

Windows XP SP3 Benefits

Here are the key benefits :

N E T W O R K I N G

  • It will make xp more secure with all the new updates and patches.
  • It has all the performance, stability and security updates released between SP2 and when it is released. This creates a new starting point when deploying new pcs or installing upgrading hard drives. The time downloading updates will be less after sp3 is installed. There have been hundreds of updates between sp2 and sp3.
  • Network Access Protection (NAP) which has been available on Windows Vista and Server 2008, will be a part of XP when sp3 is released. This feature is useful in a corporate environment and allows network administrators more granular control over client pcs requiring network access.
  • SP3 will improve black hole router detection (a black hole router is one that silently discards packets). Again this is most useful in a corporate environment.

S E C U R I T Y

  • The Microsoft Kernel Mode Cryptographic Module will be introduced to XP in SP3. This module will allow your pc different ways to encrypt/hide your data information from prying eyes. Thus information can be safely moved or stored.
  • The Control Panel Security Options will have more descriptive text explaining security settings and aids to prevent incorrect security settings.

M I S C E L L A N E O U S

  • As mentioned above, HD audio support is a new feature. This will be beneficial to gamers with the appropriate hardware.
  • When performing a complete operating system install with sp3 as the base, you will no longer need to enter a product key before xp is usable. In other words, you will be able to use xp without a product key. You will still have to activate xp and a product key is needed for that.
