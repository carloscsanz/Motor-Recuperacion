Re: wcf call timeout
You should increase you maxReceivedMessageSize value on client side. Example maxReceivedMessageSize="100000000" approximately 100 Mb.

You can also calculate the size of object, which you return from WCF-service call on server side.
Example:

public object ServiceMethod(.....)
{
...
object res = ...

System.IO.MemoryStream stream = new System.IO.MemoryStream();
BinaryFormatter objFormatter = new BinaryFormatter();
objFormatter.Serialize(stream, res);
long lSize = stream.Length;
Logger.Write("Size of the Object: {0}", lSize);
return res;
}

Compare server object size and client maxReceivedMessageSize value.
