natural language processing application

    0.06 sec.
ThesaurusLegend:  Synonyms Related Words Antonyms
Noun1.natural language processing application - an application program that deals with natural language text
natural language processor
application program, applications programme, application - a program that gives a computer instructions that provide the user with tools to accomplish a task; "he has tried several different word processing applications"
disambiguator - (computer science) a natural language processing application that tries to determine the intended meaning of a word or phrase by examining the linguistic context in which it is used
