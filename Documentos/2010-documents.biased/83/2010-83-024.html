First off, re-register Word which will resolve one problem. See
http://word.mvps.org/FAQs/AppErrors/ReRegisterWord.htm

You must have set Word 2003 to save documents in format for legacy versions
which is why you are getting text colour warning (Office 2003 supports more
colours than previous versions and will be lost if you save in a previous
version format).

Finally, you need activate your version of Word 2003 with Microsoft or you
only have a trial version installed as your message clearly states you have
used it 11 times (out of a permitted 50 uses).
