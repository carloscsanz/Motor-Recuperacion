1. Installation Instructions

================================================== ==============================

1.0. Game Activation *PLEASE READ*

The game is protected using an activation-based technology. Online activation
is needed to play the game. When launching the game for the first time, you will
be prompted to enter your activation code.

If you encounter a problem during activation, you can get help via the following
web sites:

http://www.farcrygame.com/unlock
http://www.farcrygame.com/faqs

You can activate the game 5 times per PC on a maximum of 3 computers. However, if
you uninstall the game through the uninstallation procedure as documented in 1.1.4.,
you will recover the previously used activation slot.

1.2. Language Selection

The language selected at the start of the installation is the language that will
be used in-game. To change the language, please uninstall and reinstall the game.

--------------------------------------------------------------------------------
1.3. Installation

During the setup, the installation of PunkBuster on the computer will be offered.
PunkBuster is a piece of software published by Even Balance, Inc. to keep cheaters
at bay and provide a more enjoyable gaming experience during online multiplayer games.

While PunkBuster is optional, it will not be possible to play multiplayer games
on PunkBuster enabled servers without it. To install PunkBuster manually, download
and run PBSetup from the Far Cry 2 support page hosted by EvenBalance here:

http://www.evenbalance.com/index.php...rt-farcry2.php

For more detailed information on PunkBuster, please consult EvenBalance's support page.

--------------------------------------------------------------------------------
1.4. Uninstallation

Far Cry® 2 can be uninstall from the Add/Remove Programs icon located in the
Control Panel (Start->Settings->Control Panel). Locate the Far Cry® 2 entry and
click on the Change/Remove button, then follow the instructions on-screen to
uninstall the game.

Note: The PunkBuster services are not automatically removed when uninstalling
Far Cry® 2 because the PunkBuster services can still be used by other games on
the system.

To remove PunkBuster services, uninstall them from the Add/Remove Programs icon
on the Control Panel. Or re-run the setup (pbsvc_fcx.exe) located on the game disc
in the Support\PunkBuster folder and follow the instructions on screen.

--------------------------------------------------------------------------------
1.5. User-created map shell extension

You can also install a shell extension that will allow to get detailed information about
Far Cry® 2 user created maps directly from Windows Explorer. The information displayed
will include map name, author, creator, and other useful details.

Map shell extension installation procedure.

- Copy FC2MapExt.dll located in the Support\Tools folder of the installation layout into
your Windows directory. If the Windows operating system is running in 64-bit mode,
copy FC2MapExt64.dll instead.

- Inside a Command Prompt, write this command to register the extension:

For Windows 32-bit:
regsvr32 FC2MapExt.dll

For Windows 64-bit:
regsvr32 FC2MapExt64.dll

- To remove it, unregister the shell extension with this command:

For Windows 32-bit:
regsvr32 /u FC2MapExt.dll

For Windows 64-bit:
regsvr32 /u FC2MapExt64.dll

2. General Notes

================================================== ==============================
2.1. Display/Performance Settings

The Graphical Settings can directly impact the memory usage of the game. If your
computer lacks RAM memory, these settings should be lowered down accordingly.

There are three options that are mainly CPU-related: RealTree, Physics and Fire.
Those options can be used to reduce the CPU load for single core computers and
low CPU speed.

Most parts of the game have been tweaked to run well on both single and multi-core
systems. It will work best on a multi-core CPU and is highly recommended to
experience the game to its full potential.

There are some video card drivers that may cause flickering and disappearing
objects. It is important to make sure the latest available drivers are installed.
Forcing the number of pre-rendered frames to one (1) in the driver settings can
also possibly correct this problem.

2.2. Optimal Settings

When launching the game for the first time, the game will attempt to detect the
optimal settings for your system by default. If the game is unable to fully
identify your computer, then the optimal setting is going be medium or high.

--------------------------------------------------------------------------------
2.3. Saved Games

Windows XP users:

1. Click the "My Documents" icon on your desktop.
2. Open the "My Games" folder.
3. Open the "Far Cry 2" folder and your saved games are stored in the
"Saved Games" sub-folder.

Windows Vista users:

In Windows Vista, the saved games are stored in a folder that can be accessed by
right-clicking on the game's icon in the Game Explorer and selecting "Saved Games."

You can also browse to the folder below by clicking the "Start" button, open the
"My Computer" window in the Start Menu:

C:\Users\<your name>\Documents\My Games\Far Cry 2\Saved Games

3.3. Dedicated Server

A simple interface to launch a dedicated server for Far Cry® 2.

Server Tab

All server settings can be set from here. Choose what kind of server you want
and start it! Take note that the "MapCycle" can be set using a more user friendly
interface, just click the button next to it when selected. Before starting the
server, you can save the current configuration. You can also load a previously
saved or a recent configuration. The latest saved or loaded configuration will be
used the next time you start the launcher, otherwise, the default configuration
will be used.

Console Tab

Once a server is started, this is where you can see whats happening. You can
issue commands in the bottom text box (using Tab will auto complete your command)
or use a button at the right.

Players Tab

The current players score and statistics can be seen from here. On the upper left,
you get the score by player for the current match (and information about the team
and the captain if appropriate). You can tell a message to the selected player with
the "Tell" text box. You can kick him with the "Kick" button. Also, when you select
a player, all of his current statistics and all messages that he sent to everyone
will be shown at the right. At the bottom, you will get the current match type. You
can say a message to everyone using the "Say" button. And finally, you get the team
score for the current match if appropriate.

Options Tab

Some options are available for you here. You can set the font, the fore and back
color of the console. You can enable/disable log files. Every time you start a
server, the console output will be saved into a file named after the current date.
You can also add the time at which each output occurs. You can see those log files
using the "Go to logs" button or delete them using the "Delete all" button.
You can enable or disable updates made to the players tab with the last option.

4. Benchmark Tool

================================================== ==============================

4.1. Benchmark Tool

The Far Cry® 2 Benchmark Tool provides a way to test many different game settings
and analyze their impacts on game performance. The tool is built to allow the
user to queue many different game settings in individual runs.

Each run can be executed on demos provided with the tool, which are mainly camera
paths, or created by the user (playbacks). While camera paths are static and
can't be modified by the user, playbacks can be recorded in-game in the
single-player campaign (default button: "numpad8" to toggle playback capture) and
used in the benchmark tool to profile selected areas of the game.

4.2. Run Settings

Run settings are classified in 3 different groups: General, Game and
Miscellaneous settings.

4.2.1 General Settings.

- Name: Name of the current run.

- Description: Description of the current run.

- Loops: Number of times to test the run. Increasing this number will give
better average results.

- Demo Name: Name of the selected demo.

- Fixed Time Step Simulation (30 FPS): Use this if you want to force the game
to update each frame by 1/30 sec (always enabled in playback). This allows to
compare results by frame instead of by time.

- Disable Artificial Intelligence: Use this setting to disable artificial
intelligence (enemies won't react to the player/environment).

4.2.2 Game Settings

Display:

-Resolution: Screen resolution and refresh rate (active in fullscreen only).

-Anti-Aliasing: Select the desired anti-aliasing mode.

-Windowed: If enabled, the game will run in windowed mode.

-VSync: If enabled, vertical synchronization is activated.

-Direct3D 9/Direct3D 10: Select the desired Direct3D version to use in the run.

-Performance: These settings affect mainly the CPU cost.

-Fire: Quality of fire simulation.

-Physics: Quality of physic simulation.

-Real Trees: Quality of real trees simulation.

-Render Quality: These settings can be edited for Direct3D 9 and Direct3D 10
separately.

-Overall Quality: Select the desired overall quality of the run.

Note: Selecting "optimal" will automatically detect the best settings for your
current system.

-Vegetation: Quality of vegetation.

-Shading: Quality of shading (water, environment, textures).

-Terrain: Quality of terrain.

-Geometry: Geometry complexity.

-Post FX: Quality of post effects.

-Texture: Controls the texture size.

-Shadow: Quality of shadows.

-Ambient: Quality of ambient lighting.

-HDR: Enable/disable high dynamic range lighting.

-Bloom: Enable/disable light bloom.

-Load Current Game Settings: Clicking on this button will load current game
settings from your gamer profile ("GamerProfile.xml").

4.2.3 Misc. Settings

Command Line:
-Generated: Generated command line used to start the game with the selected
run settings.
-Additional: Add additional command line arguments. Generated command line
arguments can be overriden in this window.

4.3. Benchmark Results

At the end of a benchmark, results are stored in a date/time-stamped folder
found in "%PersonalPath%/My Games/Far Cry 2/Benchmarks/Results/", where
"%PersonalPath%" is usually the path to your "My Documents" folder. This folder
will keep the following files:

- ".csv" files: Benchmark results for each individual loop completed. The
first column contains the frame ID, the second column the framerate and the
last column gives the end time for each frame (ie, the end time of the last
frame is the actual duration of the loop).

- ".svg" files: Benchmark framerate graphs for loops and average/overall
results. If your browser doesn't support native SVG viewer, you can download
an SVG viewer for free from Adobe (http://www.adobe.com/svg/viewer/install/).

- "index.html": The actual benchmark report file.

The report file will display framerate results (average, minimum and maximum
framerates) for each run done (individual loops and average results for a run).
Also, an overall SVG graph for runs done on the same demo with the same real-time
and fixed time step simulation flag will be generated to easily compare the
average results of these runs.

4.4. Restrictions

- To simulate as much as possible the real conditions that a player will face
in-game (Rendering, AI, Physics, etc...), all the sub-systems are simulated.

Due to the unscripted nature of the game, it may be possible that two loops
give different results. Using many loops should give good average results at
the end.

- Avoid to record playbacks when the player is attached to an entity, such as
when driving a vehicle for instance. In these cases, the player loses focus and
the playback won't match to what was recorded!

5. Known Issues

================================================== ==============================

5.1. Installing Graphics and Motherboard Drivers

If you experience any graphical problems, please install the latest video and
motherboard drivers available for your Graphics Card. Links to the most popular
Graphics Card and motherboard manufacturers are available at the bottom of this
document.

--------------------------------------------------------------------------------
5.2. Installing Sound Drivers

If you experience any sound problems, please install the latest sound drivers
available for your sound card. Links to the most popular sound card manufacturers
are available at the bottom of this document.

--------------------------------------------------------------------------------
5.3. Installing DirectX

Far Cry® 2 requires DirectX runtime libraries from June 2008 or later to
run properly. To get the latest version of DirectX, please visit:

http://www.microsoft.com/windows/directx

DirectX is also provided on the install disc in the folder Support\DirectX and
automatically installed when you install Far Cry® 2.

If you receive an error message about missing DLL files, reinstall DirectX from
the install disc to correct the problem.

--------------------------------------------------------------------------------
5.4. Hints

* It is _not_ advisable to install the game into the Windows folder (usually
C:\Windows), a subfolder of the Windows folder, nor the root of the system
drive (usually C:\).

* It is recommended to close all other running programs before installing the
game, to reduce the risk of software conflicts.

* If using Realtek High Definition Audio, Soundmax HD Audio, or nForce Audio,
you may experience sound anomalies.

To work around the problem, reduce the Hardware acceleration slider to 'Basic'
in the Control Panel->Sounds and Audio Devices->Audio tab->Advanced->Performance
tab.

* If you experience graphical corruption, make sure your graphics driver's
control panel is _not_ configured to override application settings.

* Vibration support is only available if using the Xbox 360 controller.
