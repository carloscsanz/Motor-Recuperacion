Re: Battlefield 2 and Battlefield 2142 on Vista?
Your best bet would be dual booting vista and XP, using XP for games, but if that really isnt an option expect about 10 - 15 frames less in vista than u would get in XP.

My assumption is the 'security crap' andrew was reffering to would be:

-windows defender
-user account control (the popups windows executes to verify running a program with admin privaleges)
-setting the games to run with admin privaleges (prolly the most important one, punkbuster will boot you out if you dont, unless thats been fixed since the last time i played in vista)
-you could try disabling automatic updates while playing as well

-turning off aero glass if its on and turning off the sidebar should also be done for optimal performance.

with all of that done 2142 ran pretty choppy on my Sager 5720 on vista. I couldnt take it anymore so i dual booted. Maybe some GPU drivers and other updates have been released since that will help you though, I wish you luck.

Re: Battlefield 2 and Battlefield 2142 on Vista?
I installed Bf2142 on Vista Ultimate X64 yesterday.

No issues really, EA link downloaded and installed no issues, then downloaded the game and patches in about 20 min, they all installed no issues.

I started up the game and played, first game seemed laggy and I figured it was vista x64 and that indeed there were some performance issues. I left that sever and went to another one and everything was fine.

I think I notice a small performance hit but nothing super major, I was already playing the game at settings incredibly high for my system.

1920x1080 everything max except AA only at 2x (default)

I noticed the score board after a match is not drawn correctly eather.


I did some research and think I can fix this by turning off the desktop deal on the compatabilty tab when I boot 2142, that also may get rid of that very small performance hit that I think is there.

I say think because I have not confirmed it, the drawFps 1 command is not working for me to test my average fps, so I have to get fraps installed.

There is one more thing to note, Im comparing this to my XP install of the game, but last time I played BF2142 on XP I was using hardware sound rendering with my X-Fi soundcard. Last week I broke my X-fi when I was soldering on some new opamps. So im now using my onboard audio and using software rendering.

Its a well known fact that this uses some cpu resorces, and given bf2142s pretty poor coding, I wouldnt be supprised if thats the cause of my supposed decrease in frame rate and not vista at all.

Im going to do more testing today with various tweaks and try the diffrent compatability modes and see if anything changes. I'll insall FRAPS and do a fps check and swich over to my XP boot and see if my performance is the same or better.

Note that this is x64 Vista Ultimate, BF2142 hates x64 :P if you are using the x86 vista you should have even less issues and have no problems with it hopefully.
