    clownshoes2009:
    System.Data.SqlClient.SqlException: The data types text and nvarchar are incompatible in the equal to operator.

    It means that somewhere in your code you try to assign a text to nvarchar SQL types. This is either when you assign the parameter in the SQL command or it's at the SQL server, you try to assign a typed nvarchar value to a text column.

    This is what I can think of, you need to run in debug step into and try to execute your query directly against the SQL Server in management studio. Like this you will see if it's your SQL or you C# the issue.

    There are no magic tricks. Just debugging.
