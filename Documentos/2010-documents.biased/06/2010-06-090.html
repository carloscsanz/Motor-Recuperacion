Thank you for the useful link.

In order to make sure that all the links on the website are indexed in search engines, you can create a sitemap or site index for the website and link all the webpages and links on the website to the sitemap.

Search engine spiders find it easier to locate the links and crawl them if a sitemap is available on the website. Hence, if you have sitemap on your website and all the webpages are linked to the sitemap, most of the webpages will be indexed in search engines.

It is not necessary that all the webpages in the website will be indexed in search engines if you have a sitemap on your website. However, the chances of webpages being indexed in search engines will surely increase.
