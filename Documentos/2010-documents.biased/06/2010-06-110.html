HTML Tables

Tables allow you to arrange content distributed across a grid of columns and rows. Tables are similar to spreadsheets where cells are formed by dividing a rectangle into columns and rows. Tables have and continue to play an integral function in the layout of Web pages, however, CSS is quickly changing the use of tables for layout. You can use tables to visually arrange elements in proportion to each other.

A simple table can be created using four elements.

Elements Description
<table>....</table> Container for additional elements that specify the content for a table.
<tr>....</tr> Container for one row of cells.
<th>....</th> or <td>....</td> Container for content that is rendered inside one cell of a table. The "th" tag specifies a header cell. The "td" tag specifies a data cell in a table.

Simple Table: One column and two rows:

Psychometric Theory
Behavioral Research

HTML/XHTML Advanced Data Tables

The above table discussion has focused primarily on common elements such as table, caption, tr, th, and td. However, a complete HTML/XHTML table is defined using the following structure:

  • <table> tag.
  • <caption> tag (optional).
  • One or more groups of rows. These may consist of a header section specified by the <thead> tag, a footer section specified by the <tfoot> tag, a body section specified by the <tbody> tag. These elements are option, but if used, the table must contain at least a series of rows <tr>. The rows must contain at least one header or data cells <th> and <td> respectively.
  • One or more groups of columns specified by <colgroup> with individual columns within the group indicated by the <col> tag.
  • </table> tag

Common Table Attributes

A table and its enclosed elements support a variety of attributes for alignment, sizing, and layout. However, the "align" and "bgcolor" attributes for the "table" element are deprecated in HTML 4.01 and XHTML in favor of using style sheets. Instead of the "bgcolor" attribute, consider using the CSS "background-color" property instead. To align a table use the CSS properties, margin-left and margin-right.

  • "width" attribute: The width attribute defines the width of a table. The value is measured in pixels or percentage values. Always add the width attribute to your tables for improved rendering.
  • "border" attribute: Controls the thickness (pixels) of the border drawn around a table.
  • "cellpadding" and "cellspacing" attributes: These attributes control the space between cells in a table. The value is measured in pixels or percentage values. Cellpadding is the amount of empty space between the border of a table cell and the content of the cell. Cellspacing refers to the amount of empty space between the outer edges of each table cell.
