CPU Benchmarks
What Does The Graph Mean?
The processors shown on this graph represent the high end computing market. The blue line tracks the performance score for each processor. The red line tracks the value score for each processor. The performance score is derived from a combination of six different benchmarking programs and every day applications. The synthetic benchmark programs used to test each processor include PC Mark 05, 3D Mark 06 and Passmark. The other applications and games that were used to test the performance of each processor include Photoshop CS2, Prey, and Quake IV. The value score is derived by dividing the price of each CPU by its combined performance score. The performance scores are weighted slightly in favor of the synthetic benchmark programs to make up for the applications and games lack of optimization for multi-core CPUs. The 0 to 100 scale used in the graph allows for easy comparison between products. A processor with a score of 80, for example, will be twice as powerful as a processor with a score of 40.
