Hi Don,
   This workaround should do the trick:

Create a new file in WordPad and make the preferred font settings. Save 
this file as
DEFAULT.RTF (you can choose any supported file format here). Open an
Explorer window and navigate to the DEFAULT.RTF. Right-click the file
to enable the "Read-Only" attribute. This will protect the file
against overwriting it by accident. Now drag the file from the
Explorer window to the Start button to place a shortcut in the Start
Menu. Whenever you click this menu entry, WordPad will open with your
write-protected template file. You can edit the file and save a copy
with the File > Save As command, giving the copy a new name to fit 
circumstances.

Come to think of it, that file could just as well be named "wordpad.rtf" to
make it more intuitive in the Start menu.

Obviously for this method to work,
 the system file association for RTF must be set to Wordpad
 

    To view a list of current file extension settings:
                Start/Run/CMD   enter:  assoc | more  
    (that rarely-used symbol is  shift+backslash)  The "more" displays
    it one screen-full
    at a time; hitting space bar goes to next page.

    To save a file association list:
                 Start/Run/CMD   enter:   assoc > C:\FileAssoc.txt    
                            (choose the saving location path and name to
    suit yourself)


You can also print a list, but it will fill about ten pages!  Better to 
reformat
the file created above into two or three columns to save some paper,
