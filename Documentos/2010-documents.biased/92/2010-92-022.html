Windows XP SP3 Official Final Release Schedule in April 2008

Windows XP SP3 (Service Pack 3) is probably the last service pack for Windows XP before Microsoft stops selling the operating system on June 30, 2010 or one year after Windows 7 released. Hence, the Windows XP SP3 packs over 1000 hotfixes, patches and updates, coupled with some new features such as new activation scheme, Network Access Protection Module, Black Hole Router Detection and Kernel Model Cryptographic Module.

It’s reported by users who installed beta and release candidate test versions of Windows XP SP3 that the service pack actually improve the performance of the system too, such as faster boot up time. Thus the release of the official final version of Windows XP SP3 is much awaiting for.

There are news that the release date of Widnows XP Service Pack 3 final version has been published, and all major release date will be within April 2008. The release dates for different targeted users are listed below.

  • April 14, 2008: Support is available for the release version of Service Pack 3 for Windows XP
  • April 21, 2008: Original Equipment Manufacturers (OEM), Volume License, Connect, and MSDN and TechNet subscribers
  • April 29, 2008: Microsoft Update, Windows Update, Microsoft Download Center
  • June 10, 2008: Automatic Updates

Note: RTW on April 29 is delayed.

For the general public, the data to look for is April 29, 2008 which is the general release date for Windows XP SP3. However, it’s possible that there will be leaked version from MSDN/TechNet/Connect subscribers once the the service pack is published to the privileged customers.

Update: New release schedule for Windows XP SP3 RTM has phases of Wave1 and Wave2.
